# Piscina

Welcome to my Piscina repository! This is a collection of exercises from the 42 school introductory program, organized into C programming modules and Shell scripting.

## 📑 Table of Contents

### Main Modules

| Module | Type | Description | Exercises |
|--------|------|-------------|-----------|
| **C00-C08** | C Programming | Core C programming fundamentals and concepts | 9 modules |
| **Sh-0, Sh-1** | Shell Script | Shell scripting and Unix command-line exercises | 2 modules |
| **R-00** | Recursive Reasoning | Recursive problem-solving exercises | 1 module |

## 📂 Project Structure

### 🔤 **C Programming Modules (C00-C08)**

The C programming modules cover fundamental programming concepts progressively:

- **C00** - First C Program
  - Introduction to compilation, functions, and basic output
  - Exercises: ex00 to ex08

- **C01** - Variables & Assignments
  - Working with variables, pointers, and memory addresses
  - Exercises: ex00 to ex08

- **C02** - Strings
  - String manipulation and character arrays
  - Exercises: ex00 to ex13

- **C03** - String Functions
  - Implementing custom string comparison and concatenation functions
  - Exercises: ex00 to ex05

- **C04** - Recursive Functions
  - Understanding recursion and implementing recursive algorithms
  - Exercises: ex00 to ex03

- **C05** - Iteration, Loops & Conditionals
  - Control flow, loops, and conditional statements
  - Exercises: ex00 to ex02

- **C06** - Arrays & Sorting
  - Working with arrays and sorting algorithms
  - Exercises: ex00 to ex03

- **C07** - Function Pointers & Advanced Concepts
  - Function pointers and advanced programming techniques
  - Exercises: ex00 to ex04

- **C08** - Advanced Topics
  - Complex data structures and advanced C concepts
  - Exercises: ex00 onwards

### 🐚 **Shell Script Modules**

- **Sh-0** - Shell Scripting Basics
  - Introduction to bash and shell command-line operations
  - Exercises: ex00 to ex06

- **Sh-1** - Advanced Shell Scripting
  - More complex shell scripts and Unix utilities
  - Exercises: additional shell challenges

### 🔄 **Recursive Reasoning**

- **R-00** - Recursive Problem Solving
  - Exercises focused on understanding and implementing recursion
  - Exercises: ex00 onwards

## 🚀 Getting Started

### Prerequisites
- GCC compiler (for C programs)
- Bash shell (for shell scripts)
- Unix/Linux environment or WSL (Windows Subsystem for Linux)

### Cloning the Repository

```bash
git clone https://github.com/Pau-Narvaez-Roy/Piscina.git
cd Piscina
```

### Running Exercises

#### C Programs
```bash
cd C00/ex00
gcc -o program program.c
./program
```

#### Shell Scripts
```bash
cd Sh-0/ex00
bash script.sh
```

## 📚 Learning Path

1. **Start with C00** - Get comfortable with basic compilation and output
2. **Progress through C01-C08** - Build skills incrementally with each module
3. **Complete Sh-0 and Sh-1** - Learn shell scripting fundamentals
4. **Work on R-00** - Challenge yourself with recursive problems

## 💡 Tips & Tricks

- Each exercise builds on previous concepts - complete them in order
- Use `gcc -Wall -Wextra -Werror` for strict compilation
- Test your code thoroughly before moving to the next exercise
- Read the exercise instructions carefully
- Don't hesitate to review previous modules if you get stuck
